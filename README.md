# Crestora Vault — Downloads & Updates

Official download and auto-update channel for **Crestora Vault**, a local,
zero-knowledge password manager for Windows.

> This repository contains **only** signed installers and the update manifest —
> **no source code**. The application source is maintained privately.

## Download

Get the latest signed installer from the [**Releases**](../../releases/latest) page:

- **`Crestora Vault_<version>_x64-setup.exe`** — recommended installer (per-user, no admin).
- **`Crestora Vault_<version>_x64_en-US.msi`** — MSI alternative (Add/Repair/Remove).

All installers are code-signed as **Crestora Property Group LLC** (Azure Trusted
Signing). The in-app updater reads `latest.json` from the latest release to notify
you of and install new signed versions automatically.

## Security

Crestora Vault is zero-knowledge: your vault and master password never leave your
device and are never transmitted to Crestora or any server. See the in-app privacy
information for details.

Found a security issue? Email **crestoralabs@gmail.com**.
